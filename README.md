# Nexa Full-Stack Project

## Overview
Nexa is a comprehensive full-stack solution designed for efficient data management and monitoring. The system combines a powerful backend architecture with a highly intuitive, visually rich dashboard, providing a seamless experience for tracking and analyzing data.

## Features
* Backend Infrastructure: A robust server built with Node.js and Express to handle business logic and API requests securely.
* Interactive Dashboard: A dynamic frontend built with React, offering real-time data visualization and clear, organized monitoring tools.
* Integrated Workflow: A cohesive structure that bridges data processing with user-facing analytics.

## Technologies Used
* Frontend: React, JavaScript, CSS
* Backend: Node.js, Express
* Database: mySQL

## Project Structure
nexa-project/
├── server/          # Backend API and server logic
└── client/          # Frontend dashboard and UI components

## Setup Instructions

### Prerequisites
* Ensure Node.js is installed on your local machine.

### Installation Steps
1. Clone the repository:
   git clone https://github.com/odelia50872/nexa-server-proj.git

2. Setup the Server:
   cd server
   npm install
   npm start

3. Setup the Client:
   cd ../client
   npm install
   npm start
